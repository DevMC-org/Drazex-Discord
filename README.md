Discord-bot for the project community [devmc.org](https://devmc.org/), 
designed to automate administrative tasks, notifications 
and other functionality related to the functioning of the community

## Used stack 
- **Kotlin** 
- **Spring Boot** 
- **Spring Webflux**
- **Discord4J**
- **RSocket**
